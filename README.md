# AwesomeScripts

Special Repo for Custom Scripts and Programs of Various Programming Languages.

## Scripts Includes

- Ignore Templates
  - gitignore
  - gitattributes
  - gitconfig
  - dockerignore
  - npmrc
  - nvmrc

- Github Templates
  - Readme
  - Contributing
  - Security
  - Code of Conduct
  - Pull Request Template
  - Issue Templates
